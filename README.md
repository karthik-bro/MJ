# MJ - Personal AI Assistant (Jarvis)

MJ is a Python-based personal AI assistant inspired by Jarvis.
It can perform tasks like responding to voice commands, opening applications, taking notes, and more.

---

## Features

-  Voice command recognition
-  Text-to-speech response
-  Open websites (Google, YouTube, etc.)
-  Notes feature (save and read notes)
-  Time and basic utilities
-  Smart command handling

---

## Tech Stack

- Python
- SpeechRecognition
- pyttsx3 (Text-to-Speech)
- datetime
- webbrowser
- os

---

##  Project Structure

MJ/
│── jarvis.py        # Main assistant logic
│── notes.txt       # Stores user notes
│── README.md       # Project documentation

---

## ⚙️ Installation

1. Clone the repository:

git clone https://github.com/karthik-bro/MJ.git

2. Navigate to the project folder:

cd MJ

3. Install required libraries:

pip install speechrecognition pyttsx3 pyaudio

4. Setup Weather API

i. Go to: https://openweathermap.org/api

ii. Create an account

iii. Generate API key

iv. Replace in code:

## api_key = "YOUR_API_KEY"
---

## ▶️ Usage

Run the assistant:

python jarvis.py

Then speak commands like:

- "Open YouTube"
- "What is the time?"
- "Take a note"
